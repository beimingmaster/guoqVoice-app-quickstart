# guoqVoice-app-quickstart
 - 使用方法：
   - 录制可以使用steamlit run recordvoice.py
   - 分析可以使用groq voice(其中run.py是直接转录，run_translate.py是统一翻译成英语）
   - 需要在本地生成一个.env文件，然后配置GROQAPI_KEY，其中GROQAPI_KEY在console.groq.com获得.
